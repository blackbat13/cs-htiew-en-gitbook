# Solution 2

## Exercise

Write a program according to the specification below.

### Specification

#### Input

* $$a, b$$ - two integers

#### Output

* Sum of $$a$$ and $$b$$ 

## Solution

```python
a = int(input("Input first number: "))
b = int(input("Input second number: "))

result = a + b

print(a, "+", b, "=", result)
```
