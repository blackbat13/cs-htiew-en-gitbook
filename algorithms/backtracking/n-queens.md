# N-queens problem

## Problem description

### Specification

#### Input

* $$n$$ - liczba naturalna, liczba królowych do rozstawienia

#### Output

* Jedno z poprawnych ustawień $$n$$ królowych na szachownicy o wymiarach $$n\times n$$, lub informacja, że takie ułożenie nie istnieje.
